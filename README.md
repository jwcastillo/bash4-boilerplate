## How To Use

It's simple:

```bash
# Clone this repository
git clone https://github.com/trimstray/bash4-boilerplate

# Go into the repository
cd bash4-boilerplate

# Edit build/metadata and run
cd build
./init.sh

# Run the app
cd ../sample
./bin/sample.sh
```
