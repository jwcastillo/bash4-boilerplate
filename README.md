# `bash4-boilerplate`

## How To Use

1. Clone this repository

```bash
git clone https://github.com/trimstray/bash4-boilerplate
```

2. Go into the repository

```bash
cd bash4-boilerplate
```

3. Edit `build/metadata` and run

```bash
cd build
./init.sh
```

4. Prepare your bash script

5. First start!

```bash
cd ../your-script.sh
./bin/your-script.sh --help
```
